# Jeby Go

This api service is designed to fetch and calculate core metrics so that boats can know if it is safe to leave harbor, and how rought the ride will be if they do.

This is a small API written in Go and is open source to anybody else who wants to build something with ocean data. NOAA documentation is confusing and responses vary from TXT files of CSVs to raw text to crazy JSON responses. This api unifies all of that data into a more comprehensible system, while also providing additional calculations.

### About
Jeby is the name of my parents first boat, we travel back and forth from Cape Code to Marthas Vineyard, and are always looking at marine forecast.
